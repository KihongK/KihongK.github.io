멀쩡한 사각형

문제: [프로그래머스] 멀쩡한 사각형
https://programmers.co.kr/learn/courses/30/lessons/62048?language=python3

정답

부족하지만 이것도 설명입니다.

전체 사각형 개수 = w * h 개

멀쩡한 사각형의 개수 = 전체 사각형 개수 - 잘린 사각형 개수

우리는 잘린 사각형 개수만 구하면 됩니다!!

프로그래머스에서 제공하는 예시를 보겠습니다.

예시)   W = 8,   H = 12,   result = 80

잘린 사각형 개수를 구하기 위해 어떤 규칙이 있는지 확인해볼까요?

대각선에 만나는 접점을 빨간색 점으로
그리고 각 접점을 기준으로 공통적으로 나타나는 부분을 파란색 선으로 표시했습니다.

이 반복적인 패턴이 몇개가 있는지 확인해보세요!!

사각형 6개로 이루어진 패턴 4개를 확인할 수 있습니다.
이 패턴 개수 그리고 패턴에서

주어진 정보 다시 확인

W   =   8   H   =   12  

패턴은 주어진 두 수의 최대공약수만큼 반복합니다.
if 최대공약수가 1이다. 그러면 전체 면적이 하나의 패턴으로 이해하시는게 좋습니다.

4개 패턴 에서 1개를 따로 뽑아 예시로 봅시다.
W   =   2   H   =   3
2 와 3의 최대공약수는 1입니다.

크레인 인형 뽑기

Posted 2020-10-24Updated 2020-10-24알고리즘 / 프로그래머스4 minutes read (About 542 words)

[프로그래머스] 위장

프로그래머스 위장 : https://programmers.co.kr/learn/courses/30/lessons/42578
input data example:
[[“yellow_hat”, “headgear”], [“blue_sunglasses”, “eyewear”], [“green_turban”, “headgear”]]
output : 5

문제 풀이
직관적인 이해를 위한 예시
상의: 5개, 하의: 3개, 신발: 2개
이때 5 * 3 * 2 를 계산한 결과가 어떤 의미를 갖는지 생각하면: 상의 하의 신발 모두 착용할때 가능한 조합이 나온다.
BUT
문제에서는 안입는 것도 조합으로 고려해야한다!!
So
상의 하의 신발에 1씩 추가한다(1은 안입을 경우를 고려하기 위함)
6 * 4 * 3 은 각 부위를 안 착용했을 경우까지 포함한 조합 수를 의미한다.
또 하나 고려해야할 것이 상의,하의,신발 모두 안 착용했을 경우는 제외해야하므로
6* 4 * 4 - 1 을 해줘야한다!!
