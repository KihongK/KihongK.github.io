colab 마크다운 이미지 삽입 방법

정리할 글이 있어 colab 으로 작성 중이었는데 google drive 마운트 하고 이미지의 경로를 복사해서 작업을 했는데
이미지가 출력되지 않아서 다른 방법을 찾아봤다.

Colab markdown 이미지 삽입방법!!

  1. google drive에 이미지를 업로드
    업로드 위치는 중요하지 않습니다 !!

  2. 업로드한 이미지의 링크 생성
    업로드한 이미지에 오른쪽 클릭 후 링크 생성!!

  3. Link 의 ID 부분만 복사!!
    d/ / blah blah
    노란색 줄로 표시한 부분만 복사합니다!!

  4. Done
    이제 복사한 Link_ID와 google.drive 링크를 합쳐서 사용하면 어디서든 호출이 가능한 링크가 만들어집니다.

최종적으로 사용할 링크 = “http://drive.google.com/uc?id=" + < Link_ID >

  1. 마크다운에 이미지 링크 입력!!
