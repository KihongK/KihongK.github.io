index 정렬 방법

캐글 EDA 필사를 하던 도중 시각화 부분에서 불편함을 느꼈다… 그래서 작성하게된 글입니다

예쁜 그래프이지만 index 값이 중구난방으로 되어있어 좀 아쉽게 느껴졌습니다.
만약 index 값이 R1, R2, R3, R4, …, R9, R10, R11, …, R20 이라면 어떻게 정렬이 될까요?

단순하게 sns.countplot(x, data) 를 사용하면 먼저오는 순서대로 index 처리가 되기때문에 값을 조금만 수정해도 순서가 바뀌어 만들어집니다.

위 barplot의 내용은 환자의 흉부 X-ray 사진이 주어지고 그 사진에 대한 rad_id(진단한 의사 정보) 별 진단 횟수를 세는 작업이었습니다.
근데 특정 조건에 해당하는 데이터 셋을 제외하고 시각화를 하면 순서가 바뀌어 그래프 인덱스 위치를 통일 시켜야 했습니다.

  • 첫번째: rad_id 별 모든 진단 횟수
  • 두번째: rad_id 별 이상이 있다고 한 진단의 횟수 (=이상이 없는 진단은 제외한)
  • 세번째: rad_id 별 rad_id 가 진단한 X-ray 사진의 장 수 (df이 아니라 약간 다릅니다.)

그래프를 이쁘게 보기 위해 정렬하는 방법을 알아 보겠습니다.

1. sort_values() : 인덱스가 아닌 인덱스가 갖고있는 값에 초점을 두고 정렬하는 방법

2. sort_index() : 인덱스에 초점을 두고 정렬하는 방법

데이터의 변화에 따른 차이를 보기 위한 작업을 하던 중이었기 때문에 방법2 sort_index를 사용했습니다.
데이터 별 인덱스 끼리 차이를 알아볼때라면 방법1 sort_values를 사용했을겁니다.

여기서 문제점!!
위 그래프들의 인덱스를 보시면 문자열로 이루어진 것을 확인할 수 있습니다.
저는 당연히 R1, R2, R3 … R9, R10, R11, R12 이런식으로 정렬되기를 바랬습니다.
하지만 결과는?? R1, R10, R11, R12, … R2, R3, R4 이런식으로 정렬이 됐습니다.

저는 아예 리스트에 고정된 인덱스값을 주고 순서를 강제하는 방법을 사용했습니다.
혹시 다른 좋은 방법이 있으면 알려주세요 !!

결과는!! 훨씬 비교하기 쉽고 예쁘게 됐습니다. 휴ㅠ…

멀쩡한 사각형

문제: [프로그래머스] 멀쩡한 사각형
https://programmers.co.kr/learn/courses/30/lessons/62048?language=python3

정답

부족하지만 이것도 설명입니다.

전체 사각형 개수 = w * h 개

멀쩡한 사각형의 개수 = 전체 사각형 개수 - 잘린 사각형 개수

우리는 잘린 사각형 개수만 구하면 됩니다!!

프로그래머스에서 제공하는 예시를 보겠습니다.

예시)   W = 8,   H = 12,   result = 80

잘린 사각형 개수를 구하기 위해 어떤 규칙이 있는지 확인해볼까요?

대각선에 만나는 접점을 빨간색 점으로
그리고 각 접점을 기준으로 공통적으로 나타나는 부분을 파란색 선으로 표시했습니다.

이 반복적인 패턴이 몇개가 있는지 확인해보세요!!

사각형 6개로 이루어진 패턴 4개를 확인할 수 있습니다.
이 패턴 개수 그리고 패턴에서

주어진 정보 다시 확인

W   =   8   H   =   12  

패턴은 주어진 두 수의 최대공약수만큼 반복합니다.
if 최대공약수가 1이다. 그러면 전체 면적이 하나의 패턴으로 이해하시는게 좋습니다.

4개 패턴 에서 1개를 따로 뽑아 예시로 봅시다.
W   =   2   H   =   3
